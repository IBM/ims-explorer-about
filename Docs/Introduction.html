# Introduction

# Overview

The IBM IMS Explorer for VS Code extension provides a modern development experience for IMS® databases directly inside Visual Studio Code. It consolidates activities such as managing connections, exploring catalogs, editing DBD/PSB structures, and executing SQL into a unified interface. citeturn1search1

This extension is intended for:

  • IMS system programmers
  • Database administrators (DBAs)
  • Application developers working with IMS databases

# Key Features

  • Create and manage IMS projects
  • Configure and maintain IMS database connections
  • Browse IMS catalogs, DBDs, PSBs, and PCBs
  • View DDL, segment structures, and source code
  • Run SQL queries and view results interactively
  • Download query results as CSV
  • Organize connections within the workspace for team sharing citeturn1search1

# Benefits

  • Modern IDE experience inside VS Code
  • Integrated workflows for higher productivity
  • Faster iteration cycles with in-IDE browsing and querying
  • Shared workspace configuration for team collaboration
  • Centralized configuration of Java runtimes and settings
