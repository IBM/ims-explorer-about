Overview

About

IBM IMS® Explorer for VS Code Is designed for system programmers, database administrators, and application developers, IMS Explorer for VS Code extension provides built-in features to access inside Visual Studio Code IDE. By combining multiple activities into one environment, IMS Explorer for VS Code promotes modern development practices and simplifies complex IMS database development tasks. IBM IMS Explorer for VS Code is a FREE, and easily downloadable extension in the Visual Studio Code’s marketplace.
